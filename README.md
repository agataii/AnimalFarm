# AnimalFarm
Веб-приложение для учёта животных на ферме.
Есть бэкенд (ASP.NET Core 8, PostgreSQL) и фронтенд (React + TypeScript + Vite). Реализованы: справочники типов животных и пород, карточки животных с датой поступления и возрастом, взвешивания. Доступ по ролям: обычный пользователь ведёт свои данные, администратор дополнительно управляет пользователями. Регистрация с активацией по ссылке, вход по JWT.
